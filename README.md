# https-nextjs.org-learn-dashboard-app
Following the tutorial at https://nextjs.org/learn/dashboard-app

## Install dependencies
```sh
npm install
```

## Lint files
```sh
npm run lint
```

## Format files
```sh
npm run prettier
```

## Build application
```sh
npm run build
```

## Unit tests
```sh
npm test
```

## Run application
```sh
npm run dev
```

## Run end to end tests
```sh
npm run cypress
```
